# Netflix Clone

This is a Netflix clone project with Kotlin as the frontend and Node.js as the backend. It aims to replicate some of the core features of the Netflix platform, allowing users to browse through a collection of movies and TV shows, watch trailers, and manage their account.

## Features

- User authentication: Users can sign up, log in, and manage their account securely.
- Browse movies and TV shows: Users can explore a catalog of movies and TV shows available on the platform.
- Search functionality: Users can search for specific movies or TV shows.
- User preferences: Users can add movies and TV shows to their watchlist or mark them as favorites.
- Responsive design: The app is designed to work seamlessly across different devices, including desktops, tablets, and mobile phones.

## Technologies Used

### Frontend (Kotlin)
- Kotlin: Kotlin is used for building the frontend Android application.
- Android Studio: Android Studio is used as the IDE for developing the Android application.
- Retrofit: Retrofit library is used for making HTTP requests to the backend API.

### Backend (Node.js)
- Node.js: Node.js is used for building the backend server.
- Express.js: Express.js is used as the web application framework for Node.js.
- MongoDB: MongoDB is used as the database to store user data, movie/TV show information, and user preferences.
- JWT (JSON Web Tokens): JWT is used for user authentication and authorization.
- Multer: Multer middleware is used for handling file uploads, such as movie/TV show thumbnails and trailers.
